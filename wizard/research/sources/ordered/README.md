# Ordered Wizard Source Set

This folder contains the curated scientific source set for the Dungeon wizard
concept. It is ordered by how the sources support the argument:

Scope note: these notes are a research library, not the V0 requirements list.
The current Wizard V0 does not include learning goals, evaluation, debriefing,
telemetry, traceability artifacts, or pre/post tests. Sources about those topics
remain useful for the later paper and future versions.

1. `01_learning_alignment_and_escape_rooms.md` covers educational escape-room
   design, constructive alignment, riddle/puzzle structure, and debriefing.
2. `02_authoring_tools_and_generation.md` covers serious-game authoring tools,
   model-driven design, procedural generation, and mixed-initiative creation.
3. `03_evaluation_and_methodology.md` covers evaluation, learning analytics,
   meta-analysis, and design-science research methodology.
4. `04_local_prior_work.md` covers the local Dungeon evaluation thesis and how
   its automated evaluation logic transfers to the wizard.
5. `05_masterarbeit_paper_kandidaten.md` lists additional candidates from the
   local Masterarbeit literature folders.
6. `06_web_scout_additions.md` records targeted web-confirmed additions around
   platform, dashboard, LMS, and no-code authoring sources.
7. `sources.bib` contains BibTeX entries for the scientific source set.

The folder intentionally stores metadata, notes, and links rather than full PDFs.
Use DOI links or publisher pages to retrieve source documents when needed.
